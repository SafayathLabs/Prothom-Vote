# 🗳️ Prothom Vote – Bangladesh National Election 2026

> A modern, awareness-based election information web platform for Bangladesh 🇧🇩

🔗 Live Website: https://prothom-vote.vercel.app/ 

---

## 🌟 Overview

**Prothom Vote** is an informational web project designed to raise awareness about the Bangladesh 13th General Election.  
It presents key election details, countdown timer, voter importance, and civic responsibility in a clean and engaging UI.

---

## 🎯 Mission

- Strengthen democratic awareness 🗳️  
- Encourage responsible voting behavior  
- Provide clear election-related information  
- Deliver a modern, user-friendly experience  

---

## ⚡ Key Features

✨ Live Election Countdown Timer  
📅 Official Election Schedule Display  
🗳️ Importance of Voting Explained  
ℹ️ Voter Guidelines & Instructions  
📱 Fully Responsive Design (Mobile + Desktop)  
🎨 Modern UI with Smooth Visual Effects  
⚡ Lightweight and Fast Performance  

---
